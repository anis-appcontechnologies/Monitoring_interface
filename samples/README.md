# samples/

Test ELF files and example motor configuration files for development and testing.

These files are **not required at runtime** — the AMC Interface application does not
load anything from this folder automatically.

| File | Purpose |
|---|---|
| `test_motor_5.4.elf` | Example ELF for testing elf_reader symbol extraction |
| `motor_*.txt` | Example motor parameter sets for load/save dialog testing |
| `amc_waveform_*.csv` | Example exported waveform CSV for format reference |
